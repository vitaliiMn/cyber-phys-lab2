# Лабораторная работа №2: NLP на оценку 3

---

## Стек технологий
| Компонент | Версия / Описание |
|-----------|-------------------|
| **Ollama** | Локальный сервер для запуска LLM |
| **Qwen2.5:0.5B** | Языковая модель (инференс) |
| **Python 3.8+** | Среда выполнения скрипта |
| **requests** | HTTP-клиент для отправки промптов |

---

## Выполнение инженерной задачи

| № | Пункт из ТЗ | Реализация в проекте |
|---|-------------|----------------------|
| 1 | Установить ollama | Инструкция по установке для Windows/Linux/macOS приведена ниже |
| 2 | Скачать в ней Qwen2.5:0.5b | Команда `ollama pull qwen2.5:0.5b` загружает модель в кеш Ollama |
| 3 | Поднять Ollama сервер с LLM | Сервер запускается автоматически как фоновый сервис на `http://localhost:11434` |
| 4 | Написать скрипт отправки запросов | `inference_client.py` использует библиотеку `requests` для `POST /api/generate` |
| 5 | Прогнать через 10 запросов и зафиксировать вывод | В скрипте заданы 10 вопросов  |
| 6 | Составить отчёт с двумя столбцами | Автоматически генерируется `inference_report.md` (Запрос \| Ответ LLM) |
